## 🎯 Purpose

This reference maps [Hack23 ISMS STYLE_GUIDE.md](https://github.com/Hack23/ISMS-PUBLIC/blob/main/STYLE_GUIDE.md) to EU Parliament Monitor's political intelligence writing standards for multi-language analysis across 14 supported languages.

## 📄 Document Structure: ISMS → Article Structure

| ISMS Section | Political Article Equivalent | Purpose |
|:------------:|:----------------------------:|---------|
| **Header (Logo + Title)** | **Article Header** | Brand identity + article title; same badge format |
| **Badge row** | **Article metadata** | Publication date + classification + significance score |
| **Executive Summary** | **Lead paragraph** | Who/What/When/Where/Why in 2–3 sentences |
| **Purpose & Scope** | **Context section** | Why this EP event matters now |
| **Main sections with emoji** | **Analysis sections** | Classified, Risk-assessed, SWOT-framed content |
| **References** | **Evidence section** | EP document IDs, MCP tool outputs, statistics |
| **Document Control** | **Publication footer** | Metadata; related articles; workflow attribution |

---

## 📝 Writing Standards: ISMS → Analytical Depth

| ISMS Requirement | EU Political Adaptation | Rationale |
|:----------------:|:-----------------------:|-----------|
| **Active voice** | **Active voice** | "The Parliament voted..." not "A vote was taken..." |
| **Precise terms** | **Named EP terms** | "Rapporteur" not "lead legislator"; specific group abbreviations |
| **No jargon** | **EU terms explained** | "Trilogue (three-way negotiation)" on first use |
| **Consistent formatting** | **Consistent structure** | All risk scores, confidence levels in same position |
| **Cite sources** | **Mandatory EP citations** | Every claim gets EP document reference or MCP data source |
| **Version control** | **Date-stamped** | All artifacts include YYYY-MM-DD HH:MM UTC |

### Inline Confidence Format

```
"EPP will [HIGH confidence] support the Commission's climate package based on
confirmed group position (EP MCP analyze_voting_patterns, group=EPP, 2026-03)."

"Renew may [LOW confidence] split on the digital services vote —
based on French delegation statements in plenary debate."
```
