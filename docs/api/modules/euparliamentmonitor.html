    Module euparliamentmonitor

    Description

    European Parliament Intelligence Platform — npm package entry point.

    This barrel re-exports the full library surface of the EU Parliament Monitor so that other projects can consume types, MCP clients, analysis utilities, templates, generators, strategies, and language constants without duplicating code.

    Example

    import {
      EuropeanParliamentMCPClient,
      getEPMCPClient,
      ALL_LANGUAGES,
      scoreVotingAnomaly,
      generateArticleHTML,
      BreakingNewsStrategy,
      buildBreakingNewsContent,
    } from 'euparliamentmonitor';
