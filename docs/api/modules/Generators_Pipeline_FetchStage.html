    Module Generators/Pipeline/FetchStage

    Description

    MCP data-fetching pipeline stage with circuit breaker protection.

    All functions are pure with respect to I/O: they accept an explicit client argument instead of reading module-level state, making them straightforward to unit-test with a mock client.

    The CircuitBreaker prevents cascading failures when the MCP server is degraded: after CircuitBreakerOptions.failureThreshold consecutive errors the circuit opens and subsequent calls short-circuit immediately.
